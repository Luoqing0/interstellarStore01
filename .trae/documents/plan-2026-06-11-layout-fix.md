## 修改清单

### 修改 1：数量控制消失（新编码）

**文件**：`Source/MainTabWindow_星际商店_物品网格.cs` line 436-444

**根因**：`控Y = 名称Rect.yMax + 名称Rect.height + 2f`，名称Rect 被品质(+20f)/材料(+20f)/折扣(+18f) 撑高后，`控Y` 超出格子底部，`if` 条件失败。

**修复**：固定在格子底部，不依赖动态高度。

```csharp
// 旧
if (当前布局 == 布局类型.大 && 条件满足)
{
    float 控Y = 名称Rect.yMax + 名称Rect.height + 2f;
    if (控Y + 26f < rect.yMax - 内边距 + 2f)
    {
        Rect 紧凑控制 = new Rect(rect.x + 内边距 + 2f, 控Y, 可用宽 - 4f, 24f);
        绘制数量控制(紧凑控制, 价格key);
    }
}

// 新
if (当前布局 == 布局类型.大 && 条件满足)
{
    Rect 紧凑控制 = new Rect(rect.x + 内边距 + 2f, rect.yMax - 内边距 - 24f, 可用宽 - 4f, 24f);
    绘制数量控制(紧凑控制, 价格key);
}
```

### 修改 2：看板重命名 + 视觉独立（新需求）

**目标**：将"侧边栏"改为独立面板"看板"，深色报纸风（深棕/暗金），与商店蓝紫科技风形成视觉对比。

#### 2a. 文件重命名
- `Source/MainTabWindow_星际商店_侧边栏.cs` → `Source/MainTabWindow_星际商店_看板.cs`

#### 2b. 变量/常量重命名（MainTabWindow_星际商店.cs）
| 旧名称 | 新名称 |
|--------|--------|
| `侧边栏宽` | `看板宽` |
| `绘制侧边栏(侧边栏区域)` | `绘制看板(看板区域)` |
| `缓存问候文字` | 保持（语义不变） |

#### 2c. 方法重命名（MainTabWindow_星际商店_看板.cs）
| 旧名称 | 新名称 |
|--------|--------|
| `绘制侧边栏(Rect rect)` | `绘制看板(Rect rect)` |
| `绘制侧边栏区块` | `绘制看板区块` |
| `侧边栏内边距` | `看板内边距` |
| `侧边栏滚动` | `看板滚动` |

#### 2d. 颜色方案：深色报纸风
```
当前颜色（蓝紫色调，与商店相同）：
├─ 背景: (0.05, 0.08, 0.18, 0.88)  蓝黑半透明
├─ 边框: (0.30, 0.40, 0.60, 0.50)  蓝灰
├─ 标题: 购买按钮色(1.0, 0.5, 0.15)  淘宝橙
└─ 文字: (0.70, 0.75, 0.85)         浅蓝灰

改为深色报纸风：
├─ 背景: (0.10, 0.07, 0.04, 0.92)  深棕黑（羊皮纸底色）
├─ 边框: (0.30, 0.20, 0.10, 0.60)  暗金棕边框
├─ 标题: (0.85, 0.65, 0.30)         暗金色（报纸标题感）
├─ 文字: (0.82, 0.78, 0.70)         暖灰白（纸墨色）
├─ 分隔线: (0.25, 0.18, 0.10, 0.50) 暗棕分隔
└─ 区块标题色:
    ├─ 每日折扣: (1.0, 0.45, 0.1)    保持橙色（醒目）
    ├─ 星际新闻: (0.6, 0.55, 0.4)    暖灰棕
    └─ 背景故事: (0.7, 0.55, 0.3)    暖棕金
```
