## 修改详情

### 1. ThingDef 交叉引用错误（Def 引用名过时）

**根因**: 抽卡卡池 XML 中引用了 3 个已过时/大小写错误的 ThingDef 名称，RimWorld 1.6 中这些 defName 已被重命名，导致 `Could not resolve cross-reference` 红字报错。进游戏主界面即触发。

**修复**: 通过 RimSage MCP 查询原版 Def 验证正确名称后，替换全部 4 处错误引用。

| 原名称（错误） | 正确名称 | 出现位置 |
|---------------|---------|---------|
| `HerbalMedicine` | `MedicineHerbal` | 基础补给箱 Common 池 |
| `Component` | `ComponentIndustrial` | 基础补给箱 Uncommon 池 |
| `MeleeWeapon_Monosword` | `MeleeWeapon_MonoSword` | 高级装备箱 Epic 池 |
| `MeleeWeapon_Monosword` | `MeleeWeapon_MonoSword` | 豪华秘藏箱 Rare 池 |

**位置**: `Defs/抽卡_GachaDefs/星际商店抽卡卡池.xml`（第30、37、119、169行）

**验证**: MCP `get_def_details` 确认三个正确 defName 均存在于原版 Core Defs 中。

### 2. TradePriceImprovement disabled stat 报错

**根因**: `获取交易改善()` 方法遍历所有已生成的自由殖民者（`FreeColonistsSpawned`），对每个殖民者调用 `GetStatValue(StatDefOf.TradePriceImprovement)`。

`TradePriceImprovement` stat 通过 `skillNeedFactors` 关联 `Social` 技能，且 `SkillNeed.required` 默认值为 `true`。当殖民者的 Social 技能被 `TotallyDisabled`（如某些 mod 添加的特殊 pawn、奴隶、机械族殖民者等）时，`StatWorker.IsDisabledFor` 返回 true。在开发者模式（`Prefs.DevMode`）下，`GetStatValue` 会触发 `Log.ErrorOnce` 红字报错：

> Attempted to calculate value for disabled stat TradePriceImprovement

玩家点击 mod 按钮打开星际商店时即触发（因为该方法在计算交易价格改善时被调用）。

**修复**: 在调用 `GetStatValue` 前增加前置检查，跳过 Social 技能被禁用的殖民者，与原版 RimWorld `BestCaravanPawnUtility` 的筛选逻辑保持一致。

```csharp
// 修复前
for (int i = 0; i < colonists.Count; i++)
{
    float imp = colonists[i].GetStatValue(StatDefOf.TradePriceImprovement);
    if (imp > best) best = imp;
}

// 修复后
StatDef tradeStat = StatDefOf.TradePriceImprovement;
for (int i = 0; i < colonists.Count; i++)
{
    Pawn p = colonists[i];
    if (p == null || p.skills == null) continue;
    if (p.skills.GetSkill(SkillDefOf.Social).TotallyDisabled) continue;
    float imp = p.GetStatValue(tradeStat);
    if (imp > best) best = imp;
}
```

**位置**: `Source/MainTabWindow_星际商店_交易.cs` `获取交易改善()` 方法（约第112-125行）

**根因溯源**（通过 MCP 查原版源码）:
- 报错抛出点: `Source/RimWorld/StatWorker.cs:104`
- disabled 判断逻辑: `StatWorker.IsDisabledFor` (`Source/RimWorld/StatWorker.cs:1107`)
- `SkillNeed.required` 默认值: `true` (`Source/RimWorld/SkillNeed.cs:13`)
- stat 关联技能: Social (`Defs/Core/Stats/Stats_Pawns_Social.xml:88`)
