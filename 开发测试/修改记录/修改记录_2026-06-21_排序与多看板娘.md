# 修改记录 — 2026-06-21

## 修改主题
新增商品排序功能、多看板娘系统。

## 修改文件

### 新增
- `Source/StarStore_MascotDef.cs`：自定义 Def 类，全局命名空间。
- `Defs/MascotDefs/星际商店看板娘.xml`：两位示例看板娘配置。

### 修改
- `Source/MainTabWindow_星际商店.cs`：新增 `排序方式` 枚举与状态字段，PreOpen 中随机选择看板娘与问候语。
- `Source/MainTabWindow_星际商店_标题栏.cs`：新增排序按钮（名字/价格/科技 + ↑/↓）。
- `Source/MainTabWindow_星际商店_交易.cs`：`刷新物品列表` 末尾支持按当前排序方式与方向排序。
- `Source/MainTabWindow_星际商店_看板.cs`：看板娘贴图与问候语改为读取随机选中的看板娘。
- `Defs/SidebarConfigDefs/星际商店侧边栏配置.xml`：新增 `mascotDefNames` 列表。

## 设计决策
- 排序基准：名称用 `label`，价格用 `BaseMarketValue`，科技用 `techLevel`。
- 每个分类/页面独立记忆排序状态（单一状态变量）。
- 看板娘问候池独立，每次打开商店随机选一位看板娘并随机抽一条问候语；未配置时回退旧字段。
- 工程量评估：中等偏小，新增两个 Def 类 + 少量 UI 调整。

## 测试要点
- 编译成功。
- 需进游戏验证排序按钮、不同分类下排序结果、看板娘随机切换。
