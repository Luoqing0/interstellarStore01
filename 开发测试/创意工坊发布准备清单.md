---

| 检查项 | 说明 | 状态 |
|--------|------|------|
| 版本号确认 | 将 `About.xml` 中的 `modVersion` 从测试版改为正式版，如 `1.0.0` | ☐ |

| 文件编码检查 | 所有 XML、CS 文件使用 UTF-8 编码 | ☐ |
| About.xml 校验 | 确认 `name`、`author`、`packageId`、`description`、`supportedVersions` 完整正确 | ☐ |
| 预览图 | 准备 `About/Preview.png`，尺寸建议 640×360 或 1280×720 | ☐ |

## 三、创意工坊上传步骤

1. 打开 RimWorld，进入主菜单
2. 点击 **模组(Mod)** → **打开模组文件夹**
3. 确认你的模组文件夹已放入 `Mods/` 目录
4. 返回主菜单，点击 **选项(Options)** → **创意工坊(Workshop)**
5. 点击 **上传模组(Upload Mod)**
6. 选择 `星际商店文本版` 文件夹
7. 填写标题、描述、标签（如 `Mod`、`Items`、`QoL`）
8. 设置可见性：建议先设为 **仅好友可见** 或 **私有**，确认无误后再改为公开
9. 上传封面图和截图
10. 发布

- [ ] 创意工坊页面图片和文字显示正常
- [ ] 自己从创意工坊订阅一次，测试能否正常加载
- [ ] 邀请 1-2 位玩家订阅测试
- [ ] 确认评论区、讨论区已开放
- [ ] 将反馈收集方式置顶在创意工坊讨论区

---

## 五、风险提示

- Steam 创意工坊上传后会有一段时间的审核或缓存延迟
- 不要上传 `.pdb` 调试文件，虽然无害，但会增加包体积
- 确保 `packageId` 全球唯一，避免与其他模组冲突
- 第一次上传建议先用小号或设为私有，避免正式页面出错后无法撤回
